MODEL

Info

This is a Classification Model which has been Built using Keras framework from Tensorflow Library. The Model uses Sequential API from Tensorflow and has several Dense Layers. It has totally 139,777 trainable paramteres, from which 8192 params are in Layer 1, and 131328 params are in Layer 2. The model predicts the class of the customers based on the given features like CreditScore, Geography, Balance etc. The Prediction accuracy of the model is found to be 86%, which is a good result.
